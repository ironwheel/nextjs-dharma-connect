How to Generate PNG Files

Option 1: Automated (Recommended)

  1. Install ImageMagick: brew install imagemagick
  2. Run the generation script: node generate-favicons.js

Option 2: Manual Screenshot Method

  1. Open this page in your browser
  2. Take screenshots of each icon at the exact pixel sizes shown
  3. Save them with the filenames listed below
  4. Place all files in the public/ directory

Option 3: Online Converter

  1. Upload public/favicon.svg to RealFaviconGenerator.net
  2. Download the generated package
  3. Replace the files in public/ directory
